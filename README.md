## 硬件
原理图、PCB工程位于assets文件夹。需要用嘉立创EDA专业版的文件菜单导入。  
复刻教程在开源平台，元器件照着BOM表买就没问题。**用到的热成像模组：海康威视TB4117**  
这个作品涉及到较多0402元件和0.3毫米间距的引脚焊接，**难度稍大，不建议纯小白复刻**。  
视频发布时海康TB4117参考价格：290元。  
注意：如果是UVC模式还需要一个88179芯片的网卡来改，或者让卖家帮改NCM模式。
